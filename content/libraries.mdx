# Libraries

By the end of the year 2018 there was a unified way of using MobX in React world - library [mobx-react].

Then the [React Hooks] came to the light. The [mobx-react-lite] was made to support this brave new world.

It was great for several enthusiasts, but newcomers did (and still do) suffer.

**You cannot use [mobx-react-lite] with class components.**

> In not so distant future the `mobx-react-lite` might be deprecated and become a `mobx-react@7`.
>
> Don't worry, migration will be as easy as replacing one string for another.
